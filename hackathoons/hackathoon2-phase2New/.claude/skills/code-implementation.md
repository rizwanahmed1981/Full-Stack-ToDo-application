# Code Implementation

## Description
This skill enables an agent to write, refactor, and optimize code according to established patterns and best practices. It handles implementation of technical designs and ensures code quality.

## When to Use
- When implementing technical designs
- When refactoring existing code
- When optimizing code performance
- When writing new functionality

## Trigger Keywords
implement, code, implement, write, develop, build, create, refactor, optimize, improve, fix, correct

## Skill Logic
- Translate technical designs into working code
- Follow established coding standards and patterns
- Ensure code readability and maintainability
- Apply appropriate design patterns and principles
- Perform code reviews and quality checks
