# Specification Writing

## Description
This skill enables an agent to create and refine specifications for software features, components, or systems. It handles requirement gathering, constraint definition, and structured documentation creation.

## When to Use
- When creating new feature specifications
- When refining existing requirements
- When documenting architectural decisions
- When defining clear acceptance criteria

## Trigger Keywords
spec, specification, requirements, requirements gathering, define, document, describe, outline

## Skill Logic
- Extract and organize functional requirements
- Define technical constraints and limitations
- Create structured documentation with clear sections
- Establish acceptance criteria and testable outcomes
- Identify dependencies and external interfaces
