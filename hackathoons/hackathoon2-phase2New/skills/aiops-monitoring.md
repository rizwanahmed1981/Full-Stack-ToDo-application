# AIOPS Monitoring

## Description
This skill enables an agent to monitor and maintain AI systems, tracking performance, detecting anomalies, and ensuring operational stability. It handles observability and alerting for AI applications.

## When to Use
- When monitoring AI system performance
- When detecting system anomalies or failures
- When maintaining operational stability
- When generating operational reports
- When implementing alerting systems

## Trigger Keywords
monitor, monitor, observe, track, alert, alerting, performance, anomaly, stability, dashboard, metrics, logs, tracing, observability

## Skill Logic
- Collect and analyze system metrics and logs
- Detect performance degradation or anomalies
- Generate operational dashboards and reports
- Implement alerting mechanisms for critical issues
- Maintain system health monitoring
