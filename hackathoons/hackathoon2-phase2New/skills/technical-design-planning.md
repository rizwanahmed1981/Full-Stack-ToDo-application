# Technical Design Planning

## Description
This skill enables an agent to create detailed technical designs and architecture plans for software systems. It handles system decomposition, component design, and integration strategies.

## When to Use
- When designing new system architectures
- When planning component interactions
- When defining technical constraints
- When creating implementation roadmaps

## Trigger Keywords
design, architecture, system design, technical design, blueprint, plan, roadmap, component, module, integration

## Skill Logic
- Break down system requirements into technical components
- Define system boundaries and interfaces
- Specify data flow and communication patterns
- Outline implementation strategies and approaches
- Identify potential technical risks and mitigation strategies
