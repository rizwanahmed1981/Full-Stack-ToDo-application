# Testing and Quality Assurance

## Description
This skill enables an agent to create, execute, and maintain tests for software systems. It ensures code quality, reliability, and adherence to requirements through comprehensive testing strategies.

## When to Use
- When verifying implemented functionality
- When ensuring code quality standards
- When validating system reliability
- when performing regression testing
- when creating test plans and cases

## Trigger Keywords
test, testing, quality, quality assurance, qa, validation, verification, assert, check, validate, regression

## Skill Logic
- Develop comprehensive test plans and cases
- Execute automated and manual tests
- Identify and document defects
- Ensure test coverage meets requirements
- Monitor quality metrics and reporting
